# POS Application Summary

## Project Overview
**Name**: POS System (Retail/Cafe)
**Goal**: Create a high-fidelity, premium Point of Sale dashboard with Sidebar, Header, and Main Content area.
**Design Reference**: Clean, Modern, Card-based dashboard (see reference image).

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Vanilla CSS (Variables, Flexbox/Grid)
- **Icons**: Lucide React
- **Font**: Inter (Google Fonts)

## Key Components
1.  **Sidebar**: Dark theme, rounded edges, navigation links.
2.  **Dashboard**:
    - Welcome Section
    - Stats Cards (Revenue, Orders, Products Sold)
    - Product List / Recent Transactions
    - Sales Performance Chart

## Design System
- **Background**: `#F5F7FA`
- **Sidebar**: `#1F1D2B`
- **Accent**: `#6C5DD3`
- **Border Radius**: `16px` - `20px`
- **Shadows**: Soft, diffused
